## Tree CMS

Test task to implement tree-like CMS for categories. No libs used, only pure React TS.

## Deploy

https://tree-cms-elian-cheng.netlify.app/

## Install instructions:

```bash
1. git clone https://github.com/elian-cheng/tree-cms.git
2. cd tree-cms
3. npm i
4. npm run dev
```

### Used technologies

- React
- TypeScript
- SCSS
